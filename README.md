# Market Basket Analysis with Python

## Overview
This project presents a comprehensive analysis of amazon customer behaviour analysis using the Amazon Customer Survey Analysis Survey. The analysis covers data cleaning, descriptive behaviour analysis, customer segmentation and profiling, recommendation and review insights and visualization.

## Problem Statement
-	What products or categories are frequently purchased together, indicating cross-selling opportunities? 
-	How do customer purchasing patterns vary across demographics or regions? 
-	Can we segment customers based on buying behaviour to target personalized 
-	promotions? 
-	Which products drive repeat purchases or high customer loyalty?
